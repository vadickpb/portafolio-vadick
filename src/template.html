Sistema de Historias Clínicas Prevengo

Sistema web diseñado para la administración de las historias clínicas en la Clínica Prevengo, Donde se agrega pacientes, los cuales pasan a triaje con una enfermera y posteriormente a atención con un médico. Una vez finalizada la atención se generan las historias clínicas con los datos y tratamientos de cada paciente, además de generar una receta que se le es entregada al paciente.
Este sistema se desarrollo en el lenguaje de programación php y Javascript con el framework JQUERY. Además de usar la plantilla AdminLTE.
